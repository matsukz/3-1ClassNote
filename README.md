# テスト用まとめ<br>

## セル関連

* セルの選択

    * セル（A1）選択するだけの基本構造<br>
        ```VBA:A1の選択
        Range("A1")
        ```

    * セル（A1からC3・D5）を選択する
        ```VBA:選択応用
        Range(A1:C3,D5)
        ```
        後述の文字入力や色変更にも使用できます。

    * セル（A1）に文字を入力する
        ```VBA:A1に文字を入力
        Range("A1")=100
        Range("A1")="あほばかまぬけ"
        ```
    * セル（A1） の色を変更する
        ```VBA:セル色変更
        Range("A1").Interior.ColorIndex=色コード
        ```
    * セル（A1）の文字の色を変更する
        ```
        Range("A1").Font.ColorIndex=色コード
        ```
    色コードについて<br>
    | 赤  | 青  | 黄  | 緑  |
    | --- | --- | --- | --- |
    | 3   | 5   | 6   | 10  |
